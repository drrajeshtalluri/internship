ü§ñ JavaScript Logic Guide (for script.js)

Here are the steps and concepts you'll need to bring your game to life. Write your code in script.js.

  1. Step 1: Setup Your Game Variables.

    First, set up the game's initial state. You'll need a variable for the secret number. To generate one, use `Math.random()`. The formula `Math.floor(Math.random() * 100) + 1` will give you a whole number between 1 and 100.

  2. Step 2: Select the HTML Elements.

    Just like in the build-up projects, use `document.getElementById()` to create variables for all the interactive elements: the guess input (`guessInput`), the guess button (`guessButton`), the feedback text paragraph (`feedbackText`), and the reset button (`resetButton`).

  3. Step 3: Create the Main Game Logic Function.

    Create a function, let's call it `checkGuess()`. This function will run every time the player makes a guess.

  4. Step 4: Implement `checkGuess()` Logic.
    • Read the User's Guess: Inside the function, get the value from your `guessInput` element.
    • Convert to a Number: The `.value` from an input is always text. You must convert it to a number so you can compare it. Use the `parseInt()` function for this.
    • Compare the Guess: Use an `if/elif/else` structure (in JavaScript, that's `if`, `else if`, `else`) to compare the user's number to your secret number variable.
    • Update Feedback: Based on the comparison, change the `.textContent` of your `feedbackText` element to "Too high!", "Too low!", or a winning message. You can also change the `.style.color` to red for wrong guesses and green for a correct guess.
    • Handle the Win: If the guess is correct, you should probably disable the guess input and button to stop the game, and make the "Play Again" button appear by removing its `hidden` class.
  5. Step 5: Connect Everything with Event Listeners.

    Tell your guess button to listen for a `'click'` and run your `checkGuess` function when it happens. You will also need to add a listener to the reset button to reload the page or reset the game state when clicked.
